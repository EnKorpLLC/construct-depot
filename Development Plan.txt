1. Code Changes:
   - AI implements features locally
   - Changes are committed to Git
   - Changes are pushed to GitHub
   - Vercel automatically deploys to construct-depot.vercel.app

2. Deployment Strategies:
   a. Individual Changes (When to use):
      - Critical bug fixes
      - Security updates
      - Single feature completion
      - When testing specific functionality

   b. Bundled Changes (When to use):
      - Multiple related features
      - UI/UX improvements
      - Non-critical updates
      - When changes are well-tested locally

3. Build Process:
   - Vercel attempts to build the project
   - If build fails:
     - Check Vercel build logs
     - Fix any ESLint/TypeScript errors
     - Push fixes and verify build succeeds
   - Monitor deployment status via Vercel emails

3. Testing Flow:
   - AI pushes changes and notifies when deployment should be complete
   - Human tests changes at construct-depot.vercel.app
   - Human provides feedback on functionality
   - AI makes adjustments based on feedback if needed

### Task Execution Process

## Business Vision
A platform that revolutionizes bulk building materials procurement by:
1. Enabling smaller buyers to participate in bulk purchases through order pooling
2. Creating an ecosystem connecting suppliers, contractors, and subcontractors
3. Streamlining the construction bidding process with integrated material takeoffs
4. Facilitating job management and contractor relationships

## Core Business Features

### 1. Bulk Order Aggregation
- Pool individual orders until reaching minimum order quantities (MOQ)
- Deposit collection system for partial orders
- Order status tracking and notifications
- Flexible shipping options (direct or via company warehouse)
- Real-time progress tracking toward MOQ targets

### 2. Contractor Ecosystem
- Job posting system for general contractors
- Project management tools for both general and subcontractors
- Blueprint/drawing upload system
- Automated material takeoff generation
- Bid submission and management
- Project timeline and milestone tracking

### 3. Supplier Integration
- Supplier onboarding and approval system
- Product listing with MOQ specifications
- Inventory and pricing management
- Automated order routing and shipping instructions
- Margin management and pricing rules

### 4. Material Recommendation Engine
- Blueprint analysis for material requirements
- Automated takeoff to product matching
- Smart product suggestions for subcontractor bids
- Integration with supplier catalog
